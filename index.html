Add books in bulk

Paste rows copied straight from Excel or a spreadsheet (tab‑separated), one book per line, columns in this order: Title · Author · Year · Series · Awards · Format · Read/Unread · Shelf. Only the title is required; a header row is skipped automatically, duplicates (same title and author) are ignored, and a blank Shelf is filled in automatically (Kindle/Audiobook → Digital, prize winners → Prizewinners).

Save your library

Your library lives in giles-library-data.js, kept in the same folder as this app. Changes made here live only in this open page until you save: Download data file gives you an updated copy — replace the old one and reload. The app file itself never needs re-saving. CSV export is there for spreadsheets or a future enrichment pass.
